Operations-focused data analysis project built using SQL, Google Sheets and Tableau.

## Objective
Analyze sales and operational performance data to identify inefficiencies, track KPIs, and provide business recommendations.

## Tools Used
- SQL (MySQL)
- Google Sheets / Excel
- Tableau (dashboard in progress)

## Analysis Performed
- KPI metrics (sales, profit, orders, margin)
- Regional performance analysis
- Category profitability
- Discount impact analysis
- Loss-making products
- Monthly sales trends

## Key Insights
- High discounts (>30%) lead to negative profit
- Furniture category has weak margins
- Central region underperforms
- Several products generate consistent losses

## Business Recommendations
- Reduce high discount levels
- Focus on high-margin categories
- Review pricing strategy for loss products
- Improve regional performance tracking
